# Career Search Hub

A dedicated Job Portal designed for college placement cells to help students seamlessly explore, filter, and track job opportunities. This platform bridges the gap between students and career opportunities by providing a streamlined interface to manage the job hunt.

---

## 🚀 Key Features

* *Job Listings Integration:* Displays active job openings fetched dynamically from a structured data source (data.js).
* *Advanced Search & Filters:* Allows students to quickly find relevant roles by filtering job openings by *Job Role* and *Location*.
* *Detailed Job Views:* Clear, dedicated pages providing complete information on job descriptions, eligibility criteria, and CTC.
* *Save & Bookmark:* Enables students to save interesting job postings to a personalized list for quick access and later application.

---

## 🛠️ Tech Stack

* *Frontend:* React.js, JavaScript (ES6+), HTML5, CSS3
* *Data Source:* Local JSON / JavaScript Static Data (data.js)
* *Package Manager:* npm

---
